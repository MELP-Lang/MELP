## Where to Get Help

### 💬 GitHub Discussions (Preferred)

For questions, ideas, and general discussions, please use **GitHub Discussions**:

👉 [github.com/AcarKodlama/melp/discussions](https://github.com/AcarKodlama/melp/discussions)

Categories:
- **Q&A** — Language usage, installation, general questions
- **Ideas** — Feature proposals and language design discussions
- **Announcements** — New releases and important updates
- **Introductions** — Welcome to the community

### 🐛 GitHub Issues

Open an issue **only** for:
- Compiler bugs
- Documented missing features

> ⚠️ Please do not open issues for general questions — use Discussions instead.
