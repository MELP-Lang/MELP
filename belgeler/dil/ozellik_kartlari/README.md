# belgeler_v2/dil/ozellik_kartlari/ — Özellik Kartları Dizini

Bu dizin MELP dil özelliklerinin kart formatındaki belgelerini içerir.

## Orijinal Kartlar

Orijinal özellik kartları şurada bulunur:

```
belgeler/language/feature_cards/
├── ENUM.md                    Stage 1
├── PATTERN_MATCHING.md        Stage 1
├── GENERICS.md                Stage 1
├── SCOPE_NAMED.md             Stage 1
├── EXIT_IF.md                 Stage 1
├── DEBUG_BLOCK.md             Stage 0
├── DEBUGGER.md                Stage 0
├── RESULT_TYPE.md             Stage 1
├── OPTION_TYPE.md             Stage 1
├── FFI.md                     Stage 1
├── DEFER.md                   Stage 1
├── STRING_INTERP.md           Stage 0
├── OWNERSHIP.md               Stage 2 (tasarım)
└── ... (~70 kart)
```

## Kart Formatı

Her özellik kartı şu bölümleri içerir:
1. **Genel Bakış** — 1-2 cümle tanım
2. **Sözdizimi** — İngilizce canonical örnekler
3. **Türkçe** — Normalizer karşılıkları
4. **Kısıtlamalar** — Neyin çalışmayacağı
5. **Stage** — Hangi stage'de kullanılabilir

## Bu Dizinde Özet Kartlar

Bu dizin, özellik kartlarının **konsolide özet** versiyonlarını içerecektir.
Şu an orijinal kartlara yönlendirilmektedir.

→ **Birincil kaynak:** `belgeler/language/feature_cards/`
