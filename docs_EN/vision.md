# MeLP Vision, Mission, and Philosophy

MeLP is a compiled, multi-lingual system programming language designed to democratize low-level programming through natural language syntax, combining readability, safety, and performance.

## Vision

To make low-level software development more inclusive through a natural-language-first approach and bring modern systems programming tools to a broader community.

## Mission

To provide a safe and high-performance language powered by advanced compiler technologies while lowering the learning barrier and increasing developer productivity.

## Philosophy

- Syntax is close to natural language, yet compile-time safe.
- Turkish and English are first-class syntax options (Russian is in preparation).
- Progressive complexity: `hello world` <= 2 lines, while systems programming remains possible.
- The tooling ecosystem grows together with the language itself (LSP, IDE, WASM).
