# Print - MeLP Sozdizimi

**Durum:** ✅ COMPLETE (Stage1)
**Sozdizimi Surumu:** mlp1 / Stage1

---

## 📖 TEMEL SOZDIZIMI

```mlp
print(ifade)
```

`print`, kullanici tanimli degil yerlesik bir fonksiyondur. Standart ciktiya yazar ve satir sonu ekler.

---

## ✅ TAM ORNEKLER

### Ornek 1 - Sayi
```mlp
print(42)
```

### Ornek 2 - Degisken
```mlp
numeric x = 7
print(x)
```

### Ornek 3 - String Literal
```mlp
print("Merhaba, Dunya!")
```

### Ornek 4 - Ifade Sonucu
```mlp
numeric a = 3
numeric b = 4
print(a + b)
```

### Ornek 5 - String Birlestirme
```mlp
string s1 = "foo"
string s2 = "bar"
print(s1 + s2)
```

---

## ⚠️ KRITIK KURALLAR

1. `print(x)` her zaman satir sonu ekler.
2. Numeric degerler sayisal olarak yazdirilir.
3. String ifade dogrudan `print()` icine verilebilir.
4. Format-string yoktur; birlestirme kullanin.
