# Manual Test Files

This directory contains manual test files created during development sessions.

## File Organization

These are ad-hoc test files used for:
- Feature testing during development
- Debug scenarios
- Quick experiments
- LLVM backend tests

## File Types

- `*.mlp` - MELP source files
- `*.s`, `*.s.s` - Generated assembly files
- `*.o` - Object files
- `*.ll` - LLVM IR files
- Executables (no extension)

## Note

For organized test suites, see:
- `../e2e/` - End-to-end tests
- `../integration/` - Integration tests

These manual test files can be cleaned up periodically.
